User Stories:

Invigilator:
1. As an Invigilator, I want to verify identity of students, so as to avoid malpractices.
2. As an Invigilator, I want to access microphone and camera of the students, so as to avoid cheating.
3. As an Invigilator, I want to keep track of exam time, so as to abide by allocated time.
4. As an Invigilator, I want to keep track of ports, so as to avoid any connected electronic devices.

Students:
1. As a Student, I want to generate assessment report, so that I can evaluate my performance.
2. As a Student, I want to see real time status of questions attempted, so as to manage my time efficiently.
3. As a Student, I want to login to my account securly, so as to keep my data and assessment results private.
4. As a student, I want to give feedback anonymously, so as to not reveal my identity.
5. As a student, I want to read all the necessary instructions before examination, so as to avoid confusions during the exam. 
6. As a student, I want to navigating through all the questions efficiently, so as to not be restricted in any sequence.
7. As a student, at the time of any network interruptions I want to resume my assessment from where it was interrupted, so as to keep my progress stable.

Faculty:
1. As a teacher, I want to generate assessment report, so that I can evaluate student performance.
2. As a teacher, I want to generate generic assessment report, so that I can evaluate overall performance of students appearing for assessment.
3. As a faculty, I want to see feedback of students, so as to get reviews regarding questions.
4. As a faculty, I want to shuffle the questions, so as to avoid malpractices during the exam.
